# LicitaAI

Sistema web desenvolvido em Spring Boot para gerenciamento e análise de editais de licitação.

## Funcionalidades

- Cadastro de editais
- Listagem de editais
- Visualização detalhada
- Análise automatizada (simulada por IA)
- Dashboard com indicadores

## Tecnologias

- Java 21
- Spring Boot
- Spring Data JPA
- Thymeleaf
- MySQL
- Bootstrap 5

## Execução

```bash
.\mvnw.cmd spring-boot:run
